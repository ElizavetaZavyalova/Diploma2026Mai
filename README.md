Компоненты
Raspberry Pi
Определяет угол расположения дрона относительно устройства и отправляет полученные данные на сервер.
FastAPI Service
Принимает данные через REST API и сохраняет их в Redis.
Redis
Используется для быстрого хранения и передачи текущего значения угла между сервисами.
GUI Service
Получает данные из Redis и отображает направление на дрон в пользовательском интерфейсе.
Принцип работы
Raspberry Pi вычисляет угол направления на дрон.
Значение угла отправляется в FastAPI по REST API.
FastAPI сохраняет полученные данные в Redis.
GUI Service считывает значение угла из Redis и отображает его пользователю.
