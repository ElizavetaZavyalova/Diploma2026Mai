
Conv-TasNet для разделения шума дрона

Описание проекта

Данный проект посвящён применению архитектуры Conv-TasNet для задачи выделения шума беспилотного летательного аппарата (дрона) из смешанного аудиосигнала. Модель обрабатывает сигнал непосредственно во временной области, без использования спектральных преобразований, что позволяет эффективно разделять источники звука.

Цель

Цель проекта — обучить нейронную сеть выделять шум дрона из аудиозаписей, содержащих различные фоновые звуки. Полученная модель может использоваться в задачах акустического мониторинга, обнаружения беспилотных летательных аппаратов и предварительной обработки аудиоданных.

Архитектура

Модель основана на архитектуре Conv-TasNet и включает три основных компонента:

- Encoder — преобразует входной аудиосигнал в скрытое представление.
- Temporal Convolutional Network (TCN) — оценивает маску целевого источника с помощью последовательности дилатированных одномерных свёрток.
- Decoder — восстанавливает аудиосигнал шума дрона из скрытого представления.

Обучение

Для обучения использовались пары аудиозаписей:

- смешанный сигнал (шум дрона и фоновые звуки);
- целевой сигнал, содержащий только шум дрона.

В качестве функции потерь использовалась NegSI-SDR  Данная функция максимизирует сходство между предсказанным и эталонным сигналами независимо от их абсолютной амплитуды, что делает её эффективной для задач разделения аудиоисточников.

Особенности проекта

- выделение шума дрона из смешанных записей;
- использование архитектуры Conv-TasNet;
- обучение с использованием функции потерь NegSI-SDR;
- реализация на языке Python с использованием библиотеки PyTorch.

Используемые технологии

- Python
- PyTorch
- NumPy
- Torchaudio

Оценка модели

 Контроль процесса обучения осуществлялся по значению функции потерь NegSI-SDR на обучающей и валидационной выборках.
